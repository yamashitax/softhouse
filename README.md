# Softhouse

## Requirements

* [Bun](https://bun.sh)

`curl -fsSL https://bun.sh/install | bash`

## To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts testfile
```

To run tests:
```bash
bun test
```

This project was created using `bun init` in bun v1.0.20. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
